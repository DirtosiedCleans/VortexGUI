modern roblox ui lib for sigmas

## features
- modern design
- smooth animations
- works on potato executors
- auto saves settings
- anti detection
- error handling

## elements
- buttons (click stuff)
- toggles (on/off)
- sliders (pick numbers)
- dropdowns (pick options)
- color pickers
- keybinds
- notifications

## themes

- red (blood)
- green (grass)
- purple (lean)
- custom (make your own)
